# Два множники
> [e-olymp](https://www.eolymp.com/uk/problems/5)

## Постановка задачі
Яке найменше число n можна представити у вигляді добутку n = a⋅b рівно k (1≤k≤50) способами? 

Добутки a⋅b та b⋅a вважаються одним способом, всі числа натуральні.
### Вхідні дані
Одне число k.
### Вихідні дані
Виведіть число n.
## Приклад

### Вхідні дані #1
```
1
```
### Вихідні дані#1
```
1
```
### Вхідні дані #2
```
2
```
### Вихідні дані#2
```
4
```
