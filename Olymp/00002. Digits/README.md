# Цифри
> [e-olymp](https://www.eolymp.com/uk/problems/2)

## Постановка задачі
Підрахувати кількість цифр цілого невід'ємного числа n.
### Вхідні дані
Одне ціле невід'ємне число n(0≤n≤2⋅10^9).
### Вихідні дані
Виведіть кількість цифр у числі n.
## Приклад

### Вхідні дані #1
```
12343
```
### Вихідні дані#1
```
5
```
